Descripción Técnica

Nmap (Network Mapper) es una herramienta de código abierto para exploración de red y auditoría de seguridad. Se utiliza para descubrir hosts y servicios en una red, creando así un "mapa" de la red.

Cuándo se Usa

  • Reconocimiento inicial en pruebas de penetración
  • Inventario de red
  • Auditoría de seguridad
  • Monitoreo de servicios

Capacidades

  • Descubrimiento de hosts
  • Detección de puertos y servicios
  • Detección de versión de servicios
  • Detección de sistema operativo
  • Scripting con NSE (Nmap Scripting Engine)

Comandos Clave

Escaneo Básico

# Escaneo de puertos comunes
nmap 192.168.1.1

# Escaneo de todos los puertos
nmap -p- 192.168.1.1

# Escaneo con detección de versión
nmap -sV 192.168.1.1

Escaneo Avanzado

# Escaneo con scripts por defecto
nmap -sC 192.168.1.1

# Escaneo de sistema operativo
nmap -O 192.168.1.1

# Escaneo silencioso (SYN stealth)
nmap -sS 192.168.1.1

# Escaneo UDP
nmap -sU 192.168.1.1

Escaneo de Red

# Escaneo de rango de IPs
nmap 192.168.1.1-100

# Escaneo de subred
nmap 192.168.1.0/24

# Escaneo desde archivo de IPs
nmap -iL targets.txt

Ejemplos de Uso

Escaneo Completo

# Escaneo completo (puertos, versiones, scripts, OS)
nmap -A 192.168.1.100

Interpretación de Resultados

PORT   STATE SERVICE VERSION
22/tcp open  ssh     OpenSSH 7.6p1 Ubuntu 4ubuntu0.3
80/tcp open  http    Apache httpd 2.4.29 ((Ubuntu))

Estado de puertos:

  • open: Aplicación escuchando en el puerto
  • closed: Puerto accesible pero sin aplicación
  • filtered: Puerto no accesible (firewall)

Buenas Prácticas

  • Usar timing apropiado (-T0 a -T5)
  • Combinar técnicas de escaneo
  • Usar NSE para detección de vulnerabilidades
  • Guardar resultados en formato normalizado (-oA)
  • Ser consciente del ruido de red generado
